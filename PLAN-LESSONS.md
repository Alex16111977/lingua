## СИСТЕМА АГЕНТОВ-ПРОВЕРЩИКОВ

После создания каждого lesson-XX.js запускаются 4 агента проверки.
Каждый агент — это отдельная роль с конкретным чеклистом.
**Урок НЕ считается готовым, пока все 4 агента не дали "PASS".**

### АГЕНТ 1: ЛИНГВИСТ (проверка немецкого языка)

Роль: преподаватель немецкого языка, носитель.
Задача: проверить ВСЁ немецкое содержимое на грамматическую корректность.

```
ЧЕКЛИСТ ЛИНГВИСТА:

СЛОВАРЬ (vocabulary[]):
□ Каждое существительное имеет ПРАВИЛЬНЫЙ артикль (der/die/das)
  Как проверить: для каждого слова мысленно подставь в предложение.
  Частые ошибки: der Gepäck (НЕВЕРНО! → das Gepäck), die Flughafen (НЕВЕРНО! → der Flughafen)
□ Каждое существительное имеет правильный gender (maskulin/feminin/neutrum)
  gender ДОЛЖЕН совпадать с артиклем: der=maskulin, die=feminin, das=neutrum
□ Глаголы в инфинитиве (не в спряжённой форме)
□ Примеры (example) — грамматически верные немецкие предложения
□ Нет базовых A1 слов (ich, du, er, sie, sein, haben, gehen, kommen, Mann, Frau, gut, schlecht)

ТЕКСТ (text):
□ Текст совпадает с оригиналом из PDF/lingua.com (не перефразирован!)
□ Нет опечаток в немецком тексте
□ Umlauts корректны: ä, ö, ü, ß

ПЕРЕВОДЫ (sentenceTranslations):
□ КАЖДОЕ предложение текста имеет перевод (не пропущено ни одного!)
□ Переводы точные, не искажают смысл
□ Количество предложений в переводах = количество предложений в тексте

ВОПРОСЫ (questions, comprehensionQuiz):
□ Вопросы на немецком — грамматически верны
□ Ответы ДЕЙСТВИТЕЛЬНО содержатся в тексте (не выдуманы!)
□ В comprehensionQuiz: correct СОВПАДАЕТ с одним из options

FILL STORY (fillStory):
□ Каждый blank.answer — правильная немецкая форма
□ Артикли в нужном ПАДЕЖЕ:
  - nach + Dativ: dem/der/dem
  - in + Akkusativ (движение): den/die/das
  - in + Dativ (место): dem/der/dem
  - durch + Akkusativ: den/die/das
  - von + Dativ: dem/der/dem
  - auf + Dativ (место): dem/der/dem
□ Подсказки (hint) содержат правильный род и падеж
□ Текст fillStory.text грамматически сочетается с ответами
```

## ЗАЩИТА ОТ ГАЛЛЮЦИНАЦИЙ

### Правило 1: Не выдумывай — бери из источника
- Текст — ТОЛЬКО из PDF файла, НЕ перефразировать
- Вопросы comprehensionQuiz — ТОЛЬКО те, ответы на которые есть в тексте
- Словарь — ТОЛЬКО слова из текста или напрямую связанные с темой
- Примеры — ТОЛЬКО из текста

### Правило 2: Проверяй артикли по словарю
- Перед назначением артикля проверь: "der Koffer" или "das Koffer"?
- Если не уверен — проверь в тексте PDF (там артикли в контексте)
- Если не уверен — проверь в Словарь.docx из STANDART папки
- ЧАСТЫЕ ОШИБКИ (запомни!):
  das Gepäck (НЕ der!), der Urlaub (НЕ das!), das Reisebüro (НЕ die!)
  die Bordkarte (НЕ der!), der Flugsteig (НЕ das!), das Handgepäck (НЕ der!)

### Правило 3: Проверяй падежи
- fillStory.blanks — артикль ДОЛЖЕН быть в правильном падеже
- Как проверить: подставь слово в русский текст и спроси "какой падеж?"
  "Семья едет в ___" → КУДА? → Akkusativ → "den Flughafen"
  "На _____ семья регистрируется" → ГДЕ? → Dativ → "dem Flugschalter"
  "Она проходит через _____" → ЧЕРЕЗ ЧТО? → Akkusativ → "die Sicherheitskontrolle"

### Правило 4: Проверяй синонимы
- В промте synonyms: правильный вариант ДОЛЖЕН означать то же самое
- Неправильные варианты ДОЛЖНЫ менять смысл
- ТЕСТ: замени слово в предложении — смысл сохранился? Если да — это синоним.

### Правило 5: Считай предложения
- sentenceTranslations ДОЛЖЕН содержать СТОЛЬКО ЖЕ переводов, сколько предложений в тексте
- Как проверить: разбей text по точкам/восклицательным/вопросительным → посчитай → сравни

### Правило 6: Не копируй промты
- КАЖДЫЙ промт УНИКАЛЕН для данного текста
- ПОИСК: если в промте lesson-07 встречается "Flughafen" или "Müller" — это ОШИБКА
- Промт должен содержать конкретные предложения, слова, грамматику ЭТОГО текста
