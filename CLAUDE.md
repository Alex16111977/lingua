# LinguaWeg — Инструкция для AI по созданию уроков

## О проекте

Статический сайт для изучения немецкого языка (A2-B1) через AI-промты и интерактивные упражнения. Тексты из https://lingua.com/de/deutsch/lesen/. Работает offline через `file://` и на GitHub Pages.

**Структура:**
- `index.html` — оболочка сайта (CSS + JS + роутинг)
- `lessons/lesson-XX.js` — данные каждого урока (текст + словарь + 7 промтов + упражнения)
- `Источники/` — методологическая база (ОБЯЗАТЕЛЬНО изучи перед созданием промтов)

---

## ОБЯЗАТЕЛЬНО: Изучи источники перед работой

### `Источники/The_Socratic_Method.txt` (Ward Farnsworth, 2021)
Ключевые принципы для промта "Сократ":
- **Эленхус** — поиск противоречий через вопросы, не через объяснения
- **5 элементов**: вопрос-ответ, проверка согласованности, конкретные примеры, аналогии, признание незнания
- **12 правил диалога**: истина важнее победы, вызов = акт дружбы, смирение, кандор
- AI не говорит "неправильно" — спрашивает "а можем ли мы согласиться, что...?"
- Ученик сам формулирует правило своими словами

### `Источники/Конфликтующие структуры.txt` (В.А. Лефевр, 1973)
Ключевые принципы для промта "Лефевр":
- **Рефлексия** — способность встать в позицию наблюдателя к своим мыслям
- **4 элемента понимания**: П (Планшет — карта фактов), Ц (Цель — зачем текст), Д (Доктрина — как построен), Р (Решение — пересказ)
- Обучение = формирование внутренней модели текста у ученика
- Пересказ ≠ вспоминание, пересказ = реконструкция из своей модели

## ГЛАВНЫЙ ПРИНЦИП ВСЕХ ПРОМТОВ

> **СНАЧАЛА ПОКАЖИ КАК — ПОТОМ СПРАШИВАЙ.**
> Ученик уровня A2. Нельзя бросать в воду.
> Каждый промт: объяснение → образец → практика с помощью → самостоятельно.

---

## Как писать каждый промт

### 📝 Перевод + грамматика (`grammar`)

**Цель:** Ученик переводит текст по предложениям, узнаёт грамматику в контексте.

**Структура промта:**
1. Объясни формат на русском (как будем работать)
2. Покажи ОБРАЗЕЦ: разбери первое предложение по словам + грамматика
3. Дай второе предложение — ученик переводит сам
4. Мини-уроки грамматики каждые 3-4 предложения
5. Итог: что узнал + ошибки

**Правила:**
- Максимум 2 строки на правильный ответ: `Genau! Дальше: [предложение]`
- Максимум 4 строки на ошибку: ❌→✓ + 1 фраза почему
- НЕ предлагай "альтернативные варианты" когда ответ правильный
- Грамматику объясняй ТОЛЬКО когда ученик ошибся
- Интересный факт — 1 раз на 3-4 предложения (составные слова, параллели с русским)
- Темп > идеальность. Ученику интересно ДВИГАТЬСЯ.

---

### 📚 Словарь + упражнения (`vocabulary`)

**Цель:** Ученик ЗАПОМНИЛ слова и умеет ими ПОЛЬЗОВАТЬСЯ.

**5 этапов (от пассивного к активному):**
1. **Знакомство** — показываешь слова группами по 5 (артикль + перевод + пример). Ученик только читает.
2. **Викторина** — выбор A/B/C (8 вопросов, DE→RU и RU→DE)
3. **Перевод** — ученик сам вспоминает (10 слов). Существительные ОБЯЗАТЕЛЬНО с артиклем!
4. **Контекст** — вставь слово в предложение из текста (5 заданий)
5. **Мини-пересказ** — 3-4 предложения с новыми словами

**Правила:**
- Этап 1 = только показ, БЕЗ вопросов
- Быстрый темп на этапах 2-3
- Артикли обязательны: "Gepäck" без артикля = неправильно
- Адаптация: уверенный → пропускай лёгкое, слабый → больше викторины

---

### 🎭 Диалог-тренажёр (`dialogue`)

**Цель:** РАЗГОВОРИТЬ ученика. Не проверить, а вдохновить говорить.

**4 фазы (от повторения к свободе):**
1. **Повторение** — повтори 5 фраз за мной (разогрев)
2. **Викторина** — выбери A/B/C и произнеси (с юмором!)
3. **Достройка** — я начинаю, ты заканчиваешь
4. **Свободный диалог** — ученик отвечает сам

**Инструкция преподавателю:**
- Робко отвечает → упрощай, хвали больше
- Уверенно → усложняй, задавай неожиданные вопросы
- Молчит → ПРЕДЛАГАЙ варианты, не жди молча
- Скучно → уходи от шаблона, спроси что-то смешное
- Рассказал о себе → поддержи! Это золото.
- Реагируй как человек: "Oh, Mallorca! Ich war auch mal da!"
- ВСЕ вопросы в теме текста (но смежные темы допустимы)

---

### 🏛 Сократовский разбор (`socratic`)

**Цель:** Ученик САМ открывает грамматическое правило через цепочку вопросов.

**Методология (по Farnsworth):**
1. **Вопрос-ответ** — не лекция, каждый шаг через вопрос
2. **Эленхус** — при ошибке не исправляй, задай вопрос который покажет противоречие
3. **Конкретные примеры** — из текста, не абстрактные правила
4. **Русские параллели** — Akkusativ = винительный, sich = себя, Passiv = -ся
5. **Признание незнания** — "не знаю" = первый шаг к знанию

**Структура:**
- Выбери 5 предложений из текста с разной грамматикой
- Для каждого: цепочка 5-7 вопросов (наблюдение → анализ → правило)
- После каждого предложения: ученик формулирует правило СВОИМИ СЛОВАМИ
- ВСЁ общение на русском. Немецкий — только примеры из текста.

**Правила:**
- 1-2 вопроса за раз. Жди ответ.
- При ошибке → уточняющий вопрос: "А если посмотреть на окончание?"
- После 3 неудачных попыток → маленькая подсказка
- Коротко, без лекций

---

### 🔄 Синонимизация (`synonyms`)

**Цель:** Ученик узнаёт НОВЫЕ способы выразить мысль (не придумывает сам — выбирает).

**Формат: ВИКТОРИНА A/B/C (10 вопросов)**

Для каждого вопроса:
1. Оригинальное предложение из текста + перевод на русский
2. 3 варианта: 1 правильный синоним + 2 неправильных (похожих, но меняющих смысл)
3. После ответа: объяснение ПОЧЕМУ это синоним, показ ключевой пары

**Типы трансформаций в 10 вопросах:**
1-3: Замена глагола/существительного синонимом (planen→vorhaben, buchen→reservieren)
4: Синоним целого выражения
5: Aktiv → Passiv
6: Два предложения → одно с weil (глагол в конец!)
7: Коннектор deshalb (причина-следствие)
8: Перефразирование другими словами
9: Прилагательное → относительное придаточное
10: Синоним выражения (beginnen→anfangen)

**Правила:**
- Ученик ВЫБИРАЕТ, а не придумывает
- После каждого ответа — показывай пару синонимов: planen = vorhaben
- Мини-итоги каждые 3 вопроса
- Объясняй ПОЧЕМУ неправильные варианты не подходят

---

### 🪞 Пересказ по Лефевру (`lefebvre`)

**Цель:** Ученик строит "внутреннюю карту" текста и пересказывает по ней.

**Принцип: СНАЧАЛА ПОКАЖИ ОБРАЗЕЦ**

**5 ступеней:**
1. **Образец** (AI показывает): Вот первый абзац. Смотри как строю карту:
   КТО? ЧТО? ГДЕ? КОГДА? → Пересказ 3 предложениями. "Видишь? Просто!"
2. **Вместе** (абзац 2): AI начинает карту, ученик достраивает. AI даёт начало предложений.
3. **Самостоятельно** (абзац 3): Ученик сам строит карту + пересказывает. Подсказка по запросу.
4. **Полный пересказ**: 5-6 предложений по структуре (подготовка → аэропорт → полёт).
   AI даёт подсказку-структуру из 3 фаз.
5. **Рефлексия**: Что запомнил? Что было трудно? Как бы ТЫ рассказал?

**Правила:**
- НИКОГДА не бросай в воду! Всегда сначала покажи.
- Если молчит → дай НАЧАЛО предложения, а не вопрос
- Принимай пересказ если смысл верен, даже с ошибками
- Максимум 1-2 исправления за пересказ

## Интерактивные упражнения (на сайте, без AI)

Все упражнения работают на данных из `vocabulary[]`, `questions[]`, `text`, `sentenceTranslations`, `comprehensionQuiz`, `fillStory`. Код в `index.html`.

### 6 упражнений:
1. **🎯 Викторина по словам** — DE→RU / RU→DE, 4 варианта, прогресс-бар
2. **🧩 Сопоставление слов** — 2 колонки (DE голубой, RU жёлтый), клик-соединение
3. **🎨 Артикли и род** — существительное без артикля, 3 кнопки der/die/das
4. **🧱 Конструктор предложений** — собери из фрагментов, кнопка "Подсказка" (русский перевод), кнопка "Ответ" (toggle)
5. **❓ Понимание текста** — 5 вопросов multiple choice как на lingua.com
6. **📝 Вставь слово в рассказ** — русский текст с пропусками для немецких слов с артиклями. Жёлтый тёплый фон. Кнопки: "Проверить всё" и "Показать ответы" (toggle)

## Правила для данных

### vocabulary (20-30 слов):
- Уровень **A2-B1** (НЕ включать: ich, du, sein, haben, gehen, kommen, Mann, Frau)
- Существительные: ВСЕГДА `der/die/das` + поле `gender: 'maskulin'/'feminin'/'neutrum'`
- Глаголы: инфинитив, без gender
- Пример: из текста или тесно связан с темой

### sentenceTranslations:
- **ВСЕ** предложения текста должны иметь перевод (для конструктора предложений)
- Ключ = точное немецкое предложение, значение = русский перевод

### fillStory.blanks:
- Существительные: **С АРТИКЛЕМ В НУЖНОМ ПАДЕЖЕ** (den Urlaub, dem Hotel, die Bordkarten)
- Hint показывает: русский перевод + базовая форма с родом + падеж
- Глаголы/наречия: без артикля (landet, übel)

### comprehensionQuiz:
- 5 вопросов НА НЕМЕЦКОМ из текста
- 4 варианта ответа, 1 правильный
- Как на lingua.com

## Общие правила для ВСЕХ промтов

1. **Язык общения — русский.** Немецкий только в примерах и речи ученика.
2. **Сначала покажи, потом спрашивай.** Образец → вместе → сам.
3. **Краткость.** Правильный ответ = 2 строки макс. Ошибка = 4 строки макс.
4. **Не умничай.** Ученику интересно ДВИГАТЬСЯ, а не слушать лекции.
5. **Чувствуй ученика.** Уверенный → ускоряй. Слабый → помогай. Устал → заканчивай.
6. **Хвали коротко.** "Genau!" / "Super!" — без длинных похвал.
7. **Максимум 1 исправление за ответ.** Не убивай желание говорить/переводить.
8. **Темп > идеальность.** Лучше пройти весь текст с ошибками, чем застрять на 2 предложениях.
9. **Юмор приветствуется.** Учиться должно быть весело.
10. **Артикли обязательны.** Для существительных der/die/das — всегда.

## Как создать новый урок

1. Извлеки текст из PDF в `C:\Lingua\A2\STANDART\XX. Название\`
2. Извлеки словарь из `Словарь.docx`
3. Извлеки грамматику из `*Грамматика.docx`
4. Создай `lessons/lesson-XX.js` по шаблону lesson-01.js
5. Заполни: text, vocabulary (20-30 слов A2-B1), grammarFocus, questions, sentenceTranslations (ВСЕ предложения!), comprehensionQuiz, fillStory
6. Напиши 7 промтов по инструкциям выше — адаптируй под конкретный текст
7. Добавь `<script src="lessons/lesson-XX.js"></script>` в index.html
8. Протестируй: все табы, все упражнения, все промты
